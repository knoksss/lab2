# Мини-оболочка с файловыми командами на языке Python.

Проект представляет собой простую командную оболочку, схожую с PowerShell. Реализованы команды управления файлами и каталогами, а также ведётся журнал действий пользователя.
Формат ввода указан ниже, для выхода программа ожидает команду, которая выводится на экран при её запуске.

## Возможности
- Поддержка операций:
- ls [-l] [path] - отображение файлов, находящихся в директории с возможностью посмотреть файлы в другой директории(path). Команда -l выводит полную информацию о файле.
- cd [path] - возможность перехода в директорию, с поддрежкой функций ".." и "~" для перехода в родительский и домашний каталоги соответсвенно.
- cat [name] - выводит содержимое файла в консоль.
- cp [-r] [start] [finish] - копирует файл или папку из текущей директории в назначенную.
- mv [start] [finish] - перемещает файл из одной директории в другую, если они совпадают, то меняет имя файла.
- rm [-r] [name] - удаляет файл или папку, для папки поддерживается функция рекурсивного удаления.
- zip [path] [archive_name] ; tar [path] [archive_name] - архивирует папку.
- unxip [path] ; untar [path] - достаёт файлы из архива.
- grep [pattern] [path] - помогает найти строки в документе, фрагменты которой подходят под шаблон.
- history - выводит последние команды.
- все функции логируются и записываются в файл shell.log .
